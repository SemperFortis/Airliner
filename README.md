# Airliner
A console application for AP CSA. Users can purchase tickets, view purchased tickets, add funds to their account, view statistics such as spending average, spending median, and total spent for their trip.

## Requirements
- Java SDK v13 or later
- Java added to PATH

## Installation
```bash
$ git clone https://github.com/SemperFortis/Airliner.git
$ cd airliner
$ cd src
```

## Usage
```bash
$ javac App.java
$ java App
```
